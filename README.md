# Face Detection using Python

## 📌 Overview
This project is a simple real-time face detection system developed using Python and OpenCV. It captures video from the webcam and detects human faces using the Haar Cascade Classifier. A rectangle is drawn around each detected face in real time.

## ✨ Features
- Real-time face detection
- Webcam integration
- Detects multiple faces
- Draws bounding boxes around detected faces
- Lightweight and easy to use

## 🛠️ Technologies Used
- Python
- OpenCV
- NumPy

## 📂 Project Structure
```
face-detection/
│── main.py
│── README.md
│── requirements.txt
```

## ▶️ Installation

1. Clone the repository
```
git clone https://github.com/hitheshsjecmca/face-detection.git
```

2. Navigate to the project folder
```
cd face-detection
```

3. Install the required packages
```
pip install -r requirements.txt
```

4. Run the project
```
python main.py
```

## 📸 Output
The application opens the webcam and detects faces in real time by drawing a rectangle around each detected face.
